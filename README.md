# Seúl & Manhwas 🇰🇷📚

Este proyecto es una landing page informativa dedicada a explorar la sinergia entre la ciudad de Seúl, su avanzada infraestructura tecnológica y el fenómeno cultural de los Manhwas (cómics coreanos).
🎯 Objetivo

El propósito de este sitio web es ofrecer una experiencia visual e informativa para usuarios interesados en la cultura coreana moderna. Busca destacar cómo la atmósfera urbana y tecnológica de Seúl sirve de inspiración para las historias digitales (manhwas) más populares del momento.
🛠️ Tecnologías Utilizadas

Para el desarrollo de esta interfaz se han empleado las siguientes tecnologías de desarrollo web front-end:

    HTML5: Estructura semántica del contenido (header, nav, main, sections, footer).

    CSS3: Diseño visual, gestión de layouts y galerías de imágenes (vinculado mediante style.css).

    Google Maps API (Iframe): Para la visualización de la ubicación geográfica.

## 🚀 Funcionalidades

El sitio cuenta con una navegación fluida y secciones específicas:

    Navegación Interna: Menú con enlaces anclados para un desplazamiento rápido entre secciones.

    Sección Hero: Introducción visual con llamado a la acción (CTA).

    Galería Visual: Muestrario de imágenes de la arquitectura de Seúl y portadas icónicas de manhwas.

    Listado Tecnológico: Clasificación de los pilares de innovación de la ciudad.

    Recomendaciones de Manhwas: Sección organizada por géneros (Acción, Romance, Isekai).

    Mapa Interactivo: Integración de mapa para ubicar geográficamente el contexto del sitio.

## 📋 Requerimientos

Para visualizar o editar este proyecto, asegúrate de cumplir con lo siguiente:
Para Visualización:

    Un navegador web moderno (Google Chrome, Firefox, Safari o Microsoft Edge).

    Conexión a internet (necesaria para cargar fuentes externas y el mapa).

Para Desarrollo:

    Editor de código: Visual Studio Code, Sublime Text o similar.

    Activos (Assets): El código requiere una estructura de carpetas local para funcionar correctamente:

        /icons/: Debe contener el icono gear_16532131.png.

        /img/: Debe contener todas las imágenes de Seúl y las portadas de los manhwas mencionados.

        style.css: El archivo de estilos debe estar en la raíz junto al index.html.
